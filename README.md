# Disaster Response Management System (DRMS)

# Project Overview
An automated system designed for the Ethiopian Red Cross Society (ERCS) to streamline emergency reporting and resource allocation. This platform enables real-time incident tracking via Web and SMS, ensuring faster response time. 

# Tech Stack

**Backend:** Python / Django
**Database:** PostgreSQL (PostGIS for location tracking)
**Frontend:** HTML5, CSS3, JavaScript (Progressive Web App)
**Integration:** SMS Gateway for offline reporting

## Features
- **Real-time Incident Mapping:** Visualizing disaster zones.
- **Resource Management:** Tracking vehicles and medical supplies.
- **PWA Capabilities:** Offline data entry for field volunteers.
- **SMS Reporting:** Parsing emergency texts into the database.
